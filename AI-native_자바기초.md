## 1 주제/스택:  
- 강의 제목 : AI-native 자바 기초 과정
- 코딩을 직접 타이핑 하면서 문법을 배우는 것이 아니고, AI 시대에 **"코드를 타이핑하는 사람"이 아닌 "문제를 정의하고 AI와 협업하는 사람"**으로서의 자바 기초 역량을 갖춘다.
- **프롬프트 작성 → 코드 생성·리뷰 → 프로그래밍 패러다임 이해**를 단계적으로 학습한다.
- 모든 예제는 직접 코딩을 하지 않고 프롬프트를 이용해 생성하고 문법을 리뷰 하면서 개념을 이해한다.
**파트1** 환경 구축과 프롬프트 작성 이해:
- Windows에서 Antigravity 를 다운로드 하고 설치하는 과정과 기본 사용법 설명
- Antigravity AI 도구를 이용해 현재 시스템 환경을 확인 프롬프트를 제시 하고 자바(JDK)가 설치 안돼 있는 경우 프롬프트를 이용해 자바 설치 명령과 실행 방법 설명
- 빌드 도구(Maven/Gradle)가 설치 안돼 있는 경우 Antigravity AI 도구를 이용해 프롬프트를 이용해 빌드 도구 설치 명령과 실행 방법 설명
- 빌드 도구 기본 사용법 설명과 자바 프로젝트 구조 개념 설명 
- 프롬프트 기본이해 
- 좋은 프롬프트와 나쁜 프롬프트를 실행하고 결과를 비교하면서 중요성을 이해
- 기초 코드 생성을 위한 프롬프트 작성 예시 
- 요구사항 이해 및 AI를 이용한 요구사항 작성 
- AI를 이용한 PRD 문서 작성 
**파트2** 자바 기초문법: 
- 데이터의 이해, 데이터에 이름을 붙인다는 개념으로 변수 설명과 데이터의 타입 종류(기본형/참조형)와 특징을 알아본다. 
- 데이터에서 정보로, 여러 데이터를 묶어서 정보 단위로 하나의 변수로 다룬다는 개념으로 데이터 구조의 필요성 설명과 자료 구조(배열 및 컬렉션 프레임워크 - List, Map 등) 종류와 특징을 알아본다.
- 제어문, 문자열·예외 처리
- 로직이 길어 지면서 로직에 이름을 붙인다는 개념으로 메서드의 재사용성과 종류와 특징을 알아본다, 메서드 문법 이해 
**파트3** 객체지향 프로그래밍 기초:
- 콘솔 버전 고객 정보 관리 프로그램을 AI 활용해 절차적으로 작성 한 후 리뷰, 
- 이어서 객체지향적으로 업그레이드 하는 프롬프트 작성 및 코드 생성,
- 코드 리뷰, 절차적과 객체지향 프로그래밍의 차이점과 장단점을 알아본다.
- 클래스/인스턴스, 속성(필드)/메서드 클래스 문법 및 주요 특징(캡슐화, 상속, 다형성) 이해
**파트4** 인터페이스와 모듈화: 
- 인터페이스 문법 이해 및 활용
- DI(의존성 주입) 개념을 이용해 고객 정보 저장을 파일로 저장하는 기능을 모듈화 하고 이후 JDBC를 이용한 DB 저장 모듈을 추가 하는 과정을 설명 

## 2.대상 수준:  
- "AI 시대의 서사" 교육을 이수한 
- 비전공 취업준비생  

## 3. 총 시간/회차:  
- 40시간 (하루 8시간 × 5일)   

## 4. 산출 범위: 
- 커리큘럼 + 세션 상세표

## 5. 실습 환경 제약: 
- Windows 11, Gemini 3 Pro, Antigravity (로컬)

## 6. “반드시 할 수 있어야 하는 것” 3가지
- 수강 후 자바 기초문법과 객체지향 패러다임을 이해하고, 
- AI 도구를 활용하여 요구사항으로부터 코드를 생성·리뷰하며, 
- AI 도구를 활용하여 간단한 자바 프로그램을 작성할 수 있다.

## 7. 톤·수준
- **비유 중심 설명**: "AI 시대의 서사"에서 사용한 비유 스타일을 유지하여, 비전공자가 친숙하게 접근할 수 있도록 한다.
- **실습 비율 60% 이상**: 이론 설명 후 즉시 AI 도구를 활용한 실습으로 연결한다.
- **AI-first 학습**: 문법 암기가 아닌, 프롬프트로 코드를 생성하고 리뷰하며 이해하는 방식.

## 8. 전제 조건

- 수강생은 "AI 시대의 서사 v3" 교육을 이수한 상태
- Windows 11 PC에 Java 21+(또는 적절한 최신 버전), Antigravity 에디터, Gemini 3 Pro 접근 가능
- 프로그래밍 경험 없음을 전제로 하나, IT 리터러시(웹 브라우징, 파일 관리 등)는 있음
- 강의 과정에서 모든 코드는 프롬프트를 이용해 생성하고 리뷰하면서 개념을 이해한다.
