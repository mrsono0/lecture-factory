```
Day 1: 도구 장착 (Antigravity + Python + uv) → 첫 코드 실행

  ↓ "대화법을 익혔다. 이제 파이썬의 언어를 이해하자"
Day 3: 변수 → 자료구조 → 제어문 → 예외처리 → 함수 → 리스트 컴프리헨션
  ↓ "언어를 알았다. 이제 진짜 프로그램을 만들고 검증하자"
Day 4: 절차적 코드(v1) → 구조적 리팩토링(v2) → 코드 리뷰 → 테스트
  ↓ "프로그램은 만들었다. 이제 더 좋은 구조로 발전시키자"
Day 5: OOP(v3) → 캡슐화 → 상속 → DI(v4) → 최종 수료
```

**관통 프로젝트**: "고객 정보 관리 프로그램" — Day 2(PRD 설계) → Day 4(v1 절차적 → v2 구조적) → Day 5(v3 OOP → v4 DI)로 4번 진화합니다.
