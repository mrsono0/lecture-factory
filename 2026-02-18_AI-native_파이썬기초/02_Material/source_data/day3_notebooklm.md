## 1. 변수와 이름표 상자 비유

### 정의
변수(Variable)는 데이터를 담는 상자이며, 파이썬에서는 데이터에 '이름표(포스트잇)'를 붙여 해당 값을 가리키게 하는 개념[1]

### 핵심 개념
- `변수명 = 값` 형태로 사용
- 하나의 상자(변수)에 새로운 값을 다시 넣으면, 이전 값은 사라지고 마지막에 넣은 값만 기억[2]

### 실습 예시
```python
x = 10
x = 20
print(x)  # 20 — 마지막 값만 남음
```

## 2. 기본 데이터 타입 4가지

### 정의
컴퓨터가 데이터를 처리하는 방식과 메모리 할당 기준이 되는 데이터의 종류(자료형)[3]

### 핵심 개념
- 파이썬의 가장 기본적인 4가지 자료형: 정수(int), 실수(float), 문자열(str), 논리형(bool)[2]
- `type()` 함수를 통해 데이터의 타입을 확인 가능[2][4]

### 실습 예시
```python
age = 25          # int
height = 175.5    # float
name = "홍길동"    # str
is_student = True # bool
print(type(age))  # <class 'int'>
```

## 3. 타입 변환과 f-string

### 정의
타입 변환(형변환)은 데이터의 타입을 다른 타입으로 바꾸는 과정[5], f-string은 문자열 내에 변수 값을 직관적으로 삽입하는 포맷팅 기법[6][7]

### 핵심 개념
- `input()` 함수로 사용자에게 받은 입력은 무조건 문자열(str)이므로, 숫자로 계산하려면 `int()`나 `float()`으로 형변환 필요[5][8]
- f-string은 문자열 앞에 `f`를 붙이고 중괄호 `{}` 안에 변수를 삽입[6][7]

### 실습 예시
```python
birth_year = input("출생연도: ")  # str
age = 2026 - int(birth_year)      # 형변환
print(f"당신의 나이는 {age}세입니다")  # f-string
```

## 4. 리스트와 딕셔너리

### 정의
여러 개의 데이터를 하나의 변수로 묶어서 관리하기 위한 파이썬의 대표적인 컬렉션 자료구조[9][10]

### 핵심 개념
- **리스트(list)**: 대괄호 `[]`를 사용하며 인덱스(순서)를 기준으로 데이터를 저장하고 접근[9][11]
- **딕셔너리(dict)**: 중괄호 `{}`를 사용하며, 서랍장에 라벨을 붙이듯 키(Key)와 값(Value)의 쌍으로 데이터 저장. 순서 대신 키로 접근[10][12]
- 실무에서는 '리스트 안의 딕셔너리([{}, {}])' 패턴이 매우 자주 쓰임[12]

### 실습 예시
```python
fruits = ["사과", "바나나", "딸기"]
print(fruits[0])  # "사과"

student = {"이름": "홍길동", "점수": 95}
print(student["이름"])  # "홍길동"

# 리스트 안의 딕셔너리 (실무 패턴)
customers = [
    {"name": "김철수", "age": 30},
    {"name": "이영희", "age": 25}
]
```

## 5. 조건문 if/elif/else

### 정의
특정 조건이 참(True)인지 거짓(False)인지에 따라 프로그램이 각기 다른 코드를 실행하도록 흐름을 나누는 분기 제어문[6][13]

### 핵심 개념
- `if` 조건이 참이면 해당 블록을 실행, 아니면 다음 `elif` 조건을 검사, 모두 거짓일 때 `else` 블록 실행[14]
- 파이썬은 들여쓰기로 코드 블록을 구분[6]

### 실습 예시
```python
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"
print(f"학점: {grade}")  # "학점: B"
```

## 6. 반복문 for/while

### 정의
특정 코드 블록을 원하는 횟수만큼, 혹은 특정 조건이 만족되는 동안 계속해서 실행하는 제어문[15]

### 핵심 개념
- **for**: 반복 횟수가 정해져 있거나, 리스트/문자열 등 순회 가능한 객체의 요소를 하나씩 꺼내 쓸 때 사용. 주로 `range()` 함수와 함께 사용[15][17][18]
- **while**: 조건이 참인 동안 무한히 반복하며, 루프 안에서 조건을 변경하거나 `break`를 이용해 탈출 필요[16][18]

### 실습 예시
```python
# for 반복
for i in range(5):
    print(f"{i+1}번째 반복")

# while 반복
count = 0
while count < 3:
    print(f"카운트: {count}")
    count += 1
```

## 7. 예외 처리 try/except

### 정의
프로그램 실행 중 예기치 않은 오류(예외)가 발생했을 때 프로그램이 강제 종료되는 것을 막고, 안전하게 에러를 처리하는 문법[19][20]

### 핵심 개념
- 에러가 발생할 가능성이 있는 코드를 `try` 블록에 넣고, 오류가 났을 때 실행할 대처 코드를 `except` 블록에 작성[20]
- 특정 에러명(예: `ZeroDivisionError`)을 명시하여 구체적으로 대응 가능[20]

### 실습 예시
```python
try:
    num = int(input("숫자 입력: "))
    result = 100 / num
    print(f"결과: {result}")
except ValueError:
    print("숫자를 입력해주세요!")
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다!")
```

## 8. 함수 정의와 매개변수/반환값

### 정의
반복되는 로직이나 특정 작업을 수행하는 코드 묶음에 이름을 붙여 재사용할 수 있게 만든 블록(레고 블록 비유)[21][22]

### 핵심 개념
- `def` 키워드로 정의, 함수가 외부로부터 받는 입력값을 '매개변수(Parameter)'[22][23]
- 함수 내부에서 계산된 최종 결과는 `return`을 통해 호출한 곳으로 '반환(Return)' — `print()`(단순 출력)와 달리 변수에 저장하여 재사용 가능[23][24]

### 실습 예시
```python
def calculate_bmi(weight, height):
    bmi = weight / (height / 100) ** 2
    return round(bmi, 1)

my_bmi = calculate_bmi(70, 175)
print(f"BMI: {my_bmi}")
```

## 9. 내장 함수

### 정의
파이썬 설치 시 기본적으로 내장되어 있어 별도의 패키지 설치나 모듈 선언 없이 즉시 사용할 수 있는 유용한 함수들[25]

### 핵심 개념
- **화면 입출력**: `print()`, `input()`[25][26]
- **자료형 변환**: `int()`, `float()`, `str()`, `list()`[26]
- **데이터 처리**: `len()`(길이/요소 개수), `sum()`(합계), `max()`, `min()` 등[13]

### 실습 예시
```python
numbers = [85, 92, 78, 95, 88]
print(f"개수: {len(numbers)}")   # 5
print(f"합계: {sum(numbers)}")   # 438
print(f"최고: {max(numbers)}")   # 95
print(f"평균: {sum(numbers)/len(numbers)}")  # 87.6
```

## 10. 리스트 컴프리헨션 (List Comprehension)

### 정의
기존 리스트나 순회 가능한 객체를 바탕으로 간결하게 1줄로 새로운 리스트를 생성하는 파이썬 특유의 문법. AI(codex) 도구가 자주 생성하므로 구조를 읽고 해석하는 능력이 필수적[27]

### 핵심 개념
- `[표현식 for 변수 in 반복객체 if 조건]`의 형태[28][29]

### 실습 예시
```python
# 1~10 중 짝수만 제곱한 리스트
squares = [x**2 for x in range(1, 11) if x % 2 == 0]
print(squares)  # [4, 16, 36, 64, 100]
```
